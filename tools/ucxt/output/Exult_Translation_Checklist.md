## 第三部分：檢查方法

### 方法 A：使用 grep 全文搜索

```bash
# 搜尋所有 Func090B 呼叫
grep -rn "Func090B\s*\(\s*\[" d:\git\exult\tools\ucxt\output\bg_translation\zh_scripts

# 搜尋所有英文 message() 調用
grep -rn 'message\s*\(\s*"\([A-Z][A-Za-z ]' d:\git\exult\tools\ucxt\output\bg_translation\zh_scripts
```

### 方法 B：逐檔確認清單

遍歷檢查點表，確認每一項：

1. 開啟指定檔案
2. 跳到指定行號
3. 確認未翻譯內容是否存在
4. 記錄狀態（待修 / 已修 / 符合規則）

### 方法 C：使用 VS Code Find in Files

1. 按 Ctrl+Shift+F 開啟搜尋
2. 輸入搜尋模式（例如：`Func090B\(\["wait here"`)
3. 確認搜尋結果
